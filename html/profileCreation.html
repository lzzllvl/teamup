Complete Your Profile

Step 1) Select your profile type.

Entrepreneurs can create projects and invite investors and developers to their projects. Developer and Investors can find projects that they are interested in, and submit connection requests to the entrepreneurs that created the projects.

Step 2) Complete the profile requirements

Each profile type has a set of questions that must be answered before you can proceed to using TeamUp. Fill out the form below, and you're all set to go!

Step 3) Start building your team

Once you're profile has been created, it's up to you to find projects that you love, developers that work hard, and investors with a vision that aligns with yours. It's time to TeamUp, and start building your dreams.
